Project 1: EMI Sense AI - AI-Powered Loan Optimization Platform

1. Project Overview
This project is a Gradio AI web app that helps users analyze their loan details, run prepayment/refinancing simulations to save interest, evaluate their debt-to-income health score, and chat with an interactive AI financial coach.

2. AI Used
AI API used: HuggingFace API (using openai SDK; default model Qwen/Qwen2.5-7B-Instruct), Groq API, and OpenAI API
Purpose of AI: The AI is used to act as the "EMI Sense AI Coach", diagnosing the user's financial health, compiling custom action checklists ranked by interest savings, and responding to user queries about debt optimization.

3. Features
- Interactive Optimization Simulators: monthly EMI, annual bonuses, or lump-sum calculations showing saved interest/time.
- What-If Risk Scenarios & Asset Allocation: job loss buffer, rate shocks, family expenses, and debt prepayment vs. investment benchmarking.
- Professional PDF Report Compilation: instant generation of print-ready financial health reports using ReportLab with embedded graphs and AI advice.
- AI Chat Coach: interactive real-time Q&A interface with the financial coach.
